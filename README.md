# SYNTAX ERROR 👾

SYNTAX ERROR é um jogo de digitação com estética Cyberpunk Brutalism, projetado para testar sua velocidade e precisão. Destrua blocos de código caindo antes que eles atinjam o sistema! Uma experiência intensa de digitação (Typing Game) direto no seu navegador.

🌐 **Jogue agora:** [https://arthurwillers.github.io/syntax-error/](https://arthurwillers.github.io/syntax-error/)

> **Aviso importante:** Este projeto foi **totalmente vibecodado**! ✨🤖

## 🚀 Funcionalidades

### ⌨️ Gameplay Intenso de Digitação
- **Mecânica Clássica de Typing Game:** Palavras (blocos de código) caem do topo da tela e você deve digitá-las corretamente antes do impacto.
- **Progressão Dinâmica:** A velocidade (Speed) aumenta gradativamente conforme você joga, elevando o desafio.
- **Sistema de Vidas (Lives):** Erros custam caro. O jogo exibe suas vidas restantes no HUD principal.

### ⚙️ Configurações Personalizáveis
- **Dificuldade do Sistema:** Escolha entre EASY, NORMAL e HARD para ajustar a velocidade inicial e o desafio.
- **Pacotes de Linguagem:** Selecione quais linguagens de programação você quer enfrentar (misture como quiser):
  - C, Pascal, Java, Python, JS, Shell e SQL.

### 🏆 Sistema de Pontuação (Leaderboard)
- **Score Dinâmico:** Pontue ao destruir os códigos corretamente.
- **Top 5 Local:** Se você atingir um novo recorde, insira seu nickname (até 10 caracteres) para salvar sua pontuação no Leaderboard local, armazenado diretamente no seu navegador.

## 🎨 Design & Estética (Cyberpunk Brutalism)
- **Interface Brutalista:** Fontes impactantes (Outfit e Fira Code), cores de alto contraste, menus com estilo "caixa bruta" (brutal-box).
- **Efeitos Visuais Retrô-Futuristas:** 
  - Grid 3D dinâmico ao fundo simulando um ciberespaço.
  - Efeitos globais de *Scanlines* (linhas de varredura de monitores CRT).
  - *Screen Flash* (clarão na tela) ao ocorrer uma falha crítica ou impacto.
  - Chão brilhante (*ground-glow*) que indica a zona de perigo.

---

## 🛠️ Tecnologias Utilizadas

- **Estrutura:** HTML5 Semântico.
- **Estilo:** CSS Customizado focado em animações, estética Cyberpunk e efeitos visuais 3D e retrô (sem frameworks de terceiros).
- **Programação:** Vanilla JavaScript moderno (manipulação nativa do DOM, loop de jogo com `requestAnimationFrame`, uso de `localStorage` para recordes).
- **Fontes:** Google Fonts (Outfit e Fira Code para o visual dev/terminal).

## ⚙️ Como Começar / Jogar Localmente

1. Faça o clone deste repositório:
   ```bash
   git clone https://github.com/ArthurWillers/syntax-error.git
   ```
2. Abra a pasta do projeto.
3. Não é necessário nenhum servidor ou build! Basta abrir o arquivo `index.html` em qualquer navegador moderno.
4. Escolha sua dificuldade, selecione os pacotes de linguagem e clique em **INICIAR SISTEMA**.
5. Digite o mais rápido que puder!
